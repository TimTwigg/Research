# Research

Minecraft Accessability Research Project aiming to allow visually impaired people to play the game or specfic preset scenarios of the game.

## Instructions

1. Start MineCraft and create a new single-player world. The world must be:
    - In Creative mode
    - Difficuly: peaceful
    - Allow Cheats: On
    - World Type: Superflat
2. Run `main.py` to start the server and web interface.
